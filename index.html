Installation Instructrion

 

Click Install Android App on desired Release Build you want to install.(Latest is recommended)

By now you should have seen a download starting in your status bar on the top of the screen.

When the download is finished, you may find the app file in the status bar and tap on it to start installation.

You may also find the file in the Downloads app or in the Downloads folder through File Manager.


... or not?

Allow unknown sources:

By default on Android devices, installation is not allowed from "Unknown source",

in other words from sources/sites other than the Play Store

In other to install an app from AnyConnect, this must be allowed through the following setting on the device:

Settings -> Security or Applications (depends on device and Android version) -> Check the box "Install Unknown Sources/Apps"
 

This iOS installation will only work from Safari - Be sure you are using Safari

Click Install iOS App on desired Release Build you want to install.(Latest is recommended)

By then, you should have seen an iOS popup proposing to install "StreamApp".

Installation will then be performed in the background (since iOS7): check the icons on your devices's home screen


After tapping the "Install" button in the popup, nothing seems to happen, check your device's home screen:

the installation should be in progress or you should see the icon of the app somewhere.


If an "Unable to download app" popup after some time, check your internet connection,

and be sure that you are not behind any firewall that may prevent downloading .ipa files.


Sadly, iOS won't give any detailed information on what went wrong. Check below for most common insallation issues:

- Expired Provisioning profile.

- Device UDID not in the provisioning profile

- Incompatible device


Untrusted Enterprise Developer:

Starting from iOS 9, the developer has to be trusted by the device, otherwise a popup will appear and prevent using the app.


On iOS 9.0/9.1, go to Settings -> General -> Profiles -> Tap on the developer's profile, and tap on Trust.

On iOS 9.2+, go to Settings -> General -> Device Management -> Tap on the developer's profile, and tap on Trust.
